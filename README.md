# xyzepc.github.io

Source code for my academic homepage:

https://xyzepc.github.io

This site is built with the al-folio Jekyll theme and deployed with GitHub Pages.

## Local Preview

```powershell
docker compose up -d
```

Then open:

```text
http://localhost:8080
```

## Update

```powershell
git add .
git commit -m "Update homepage"
git push
```
